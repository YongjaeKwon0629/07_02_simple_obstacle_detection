# 데이터 유형 및 변환

## 📌 개요
NumPy 배열은 메모리 효율성과 빠른 연산을 위해 동일한 데이터 타입을 강제합니다.

## 🔍 데이터 타입 확인
```python
arr = np.array([1, 2, 3])
print(arr.dtype)
```

## 🔁 데이터 타입 변환
```python
arr = np.array([1.2, 2.5, 3.8], dtype='int')
print(arr)  # → [1 2 3]
```

## 데이터 타입 목록
| 코드 | 설명             |
|------|------------------|
| i    | 정수형 (integer) |
| f    | 실수형 (float)   |
| b    | 불리언 (bool)    |
| U    | 유니코드 문자열  |

## 🚀 활용도
- 머신러닝에서 메모리 최적화
- 정수↔실수 변환으로 모델 입력 데이터 정제
