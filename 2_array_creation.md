# 배열 생성 및 배열 차원

## 배열 생성
NumPy 배열은 `np.array()` 함수를 통해 생성됩니다.
리스트, 튜플 등 Python의 시퀀스를 배열로 변환합니다.

```python
import numpy as np
arr = np.array([1, 2, 3])
```

## 다차원 배열 생성
```python
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
```

## 배열 차원 확인
```python
arr.ndim  # → 1
arr2.ndim  # → 2
```

## 배열의 기본 차원
| 차원 | 설명          | 예시                           |
|------|----------------|--------------------------------|
| 0D   | 스칼라        | `np.array(7)`                 |
| 1D   | 벡터          | `np.array([1,2,3])`           |
| 2D   | 행렬          | `np.array([[1,2],[3,4]])`     |
| 3D   | 다중 행렬(텐서)| `np.array([[[1],[2]],[[3],[4]]])` |
